# examples/

Синтетический пример всех артефактов проекта на **вымышленных** данных. Реальные
выгрузки лежат в `data/` и не версионируются — здесь показан только формат.

## `team-platform/` — одна цель (чат + топик)

| Файл | Что это |
|---|---|
| `2026-W09.jsonl`, `2026-W10.jsonl` | Сырые сообщения по ISO-неделям (источник истины). Одна строка = одно сообщение. |
| `context.md` | СЛОЙ 2: живой общий контекст — backbone для `/tg-ask`. |
| `delta-log.md` | СЛОЙ 1: хронология сводок по каждой выгрузке (новое — сверху). |
| `.last_delta.json` | Манифест последней дельты: какие id/недели новые для суммаризации. |

### Формат строки `*.jsonl`

```json
{"id": 105, "date": "2026-03-03T14:22:00", "ts": 1772540520, "from": "Alice Carter",
 "from_id": 111000001, "topic": 12, "reply_to": 12, "edited": null, "fwd": null,
 "media": null, "text": "Решение: релиз v2.3 на прод в четверг..."}
```

Поля: `id` (id сообщения в чате), `date`/`ts` (UTC), `from`/`from_id` (автор),
`topic` (id топика или `null`), `reply_to` (id сообщения, на которое отвечают),
`edited` (дата правки или `null`), `fwd` (источник форварда), `media` (тип вложения
или `null`), `text`.
