Вопрос пользователя: $ARGUMENTS

Отвечай по накопленной истории чатов. Алгоритм:

## 1. Backbone

Прочитай `context.md` всех релевантных целей (`data/*/context.md`). Если вопрос
явно про один чат/тему — ограничься нужными. context.md даёт общую картину:
участники, темы, решения, открытые вопросы.

## 2. Точные факты — grep по raw

Если нужны конкретные формулировки, цифры, даты или цитаты — ищи в недельных
шардах `data/<name>/<YYYY-Www>.jsonl` (JSON-строки с полями `id, date, from,
from_id, reply_to, text`). Используй `rg`/`grep` сразу по всем неделям чата
(`data/<name>/*.jsonl`) по ключевым словам и @упоминаниям, затем читай найденные
строки и их соседей по `id`/`reply_to` для контекста треда.

Если вопрос привязан к периоду — сужай поиск до нужных недельных файлов по их
именам (`YYYY-Www`), это дешевле, чем грепать весь чат. Не загружай шарды в
контекст целиком без нужды.

## 3. Ответ

- Отвечай по существу, на русском.
- Подкрепляй фактами: указывай автора, дату и #id сообщения.
- Если в истории ответа нет — скажи прямо, не домысливай.
- Если данные могли устареть (мы не тянем правки/удаления — только новые
  сообщения), и вопрос чувствителен к этому — предупреди.
