---
description: Light-обзор всех групп/супергрупп, активных за последние N дней (N через --days)
---

Запусти обзор venv-питоном из корня проекта:

```bash
.venv/bin/python scripts/tg_overview.py            # окно по умолчанию (7 дней)
# или явно: .venv/bin/python scripts/tg_overview.py --days 14
```

Окно N дней задаётся флагом `--days` (по умолчанию 7). Скрипт пробегает диалоги
(только группы и супергруппы), читает сообщения каждого активного чата в окне N
дней и печатает Markdown-таблицу:
`Чат · Тип · Новых · Активны · Топ-авторы · Упом. меня · Ответы мне · Мои ответы`.
Результат также сохраняется в `data/_overview.md`.

Покажи пользователю итоговую таблицу. Если он попросит — можешь после этого
предложить углубиться в конкретный чат (см. `/tg-sync` для добавления его в
постоянное отслеживание или `/tg-ask` по уже накопленным).

Замечания:
- Чаты-боты с алертами дают большие «Новых» при 1 активном авторе — это шум.
- Форум-супергруппы считаются на уровне чата (по всем топикам сразу), а не по
  отдельному топику, как в `/tg-sync`.
