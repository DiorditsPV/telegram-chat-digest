---

Помоги владельцу собрать список целей для отслеживания из последних диалогов.

## 1. Выгрузка кандидатов

Запусти venv-питоном из корня проекта (N — из аргумента `$ARGUMENTS`, если задан):

```bash
.venv/bin/python scripts/tg_discover.py --last 20
```

(Подставь число из `$ARGUMENTS` вместо 20, если пользователь его указал. Если venv
нет — сначала `python3 -m venv .venv && .venv/bin/pip install -r requirements.txt`.
Если упало на отсутствии `TG_API_ID/HASH` — попроси заполнить `.env` и залогиниться.)

Скрипт берёт N самых свежих диалогов (любого типа) и пишет их **закомментированными**
целями в `config.discovered.yaml` (этот файл в `.gitignore` — содержит реальные
имена/id чатов, не коммить его).

## 2. Показать владельцу

Покажи содержимое `config.discovered.yaml` — список `[тип] Название · id` по каждому
кандидату. Поясни: всё закомментировано, парсинг не начнётся, пока цель не перенесена
в `config.yaml`.

## 3. Курирование (по желанию владельца)

Если пользователь укажет, какие чаты отслеживать, перенеси выбранные цели из
`config.discovered.yaml` в `config.yaml` (секция `targets`), раскомментировав их
(убрать `#   ` в начале строк → отступ 2 пробела). При необходимости проставь
`topic` для форум-чатов. Затем подскажи запустить `/tg-sync`.

Не переноси цели без явного решения владельца — это его выбор, что отслеживать.
